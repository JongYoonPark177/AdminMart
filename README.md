## ASP.NET Core MVC 
ASP.NET Core MVC(.NET 8.0) 쇼핑몰 관리자 페이지

<h2>🖥프로젝트 소개</h2>
무료 부트스트랩 레이아웃을 참고하여 제작한 관리자페이지 입니다.<br>
매출, 주문, 상품, 일정등을 관리할 수 있는 사이트

<h2>💾개발환경</h2>
<ul>
  <li>언어 : C# </li>
  <li>서버 : IIS </li>
  <li>프레임워크 : ASP.NET Core MVC </li>
  <li>버전 : .NET 8.0 </li>
  <li>데이터베이스 : MSSQL</li>
</ul>
