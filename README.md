## ASP.NET Core MVC 
ASP.NET Core MVC(.NET 8.0) 공부 페이지

<h2>🖥프로젝트 소개</h2>
무료 부트스트랩 레이아웃을 참고하여 제작한 관리자페이지 입니다.<br>
리스트, 업데이트, 삭제 기능등을 만들면서 ASP.NET Core MVC강의를 바탕으로 <br>
공부하면서 만든 페이지입니다.

<h2>💾개발환경</h2>
<ul>
  <li>Language : C# </li>
  <li>Server : IIS </li>
  <li>FrameWork : ASP.NET Core MVC </li>
  <li>Version : .NET 8.0 </li>
  <li>DataBase : MariaDB</li>
</ul>
